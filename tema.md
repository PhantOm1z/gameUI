🎨 Visual Style (VERY IMPORTANT)
Theme: Fantasy medieval (NO sci-fi, NO futuristic elements)
Mood: Warm, magical, cozy but heroic
Materials:
Wood, parchment, stone, gold trims
Soft glowing magic effects (subtle, not flashy)
Colors:
Earth tones (brown, beige, dark green)
Gold accents for important UI
Soft blue/purple for magic elements
Lighting:
Warm ambient light
Slight vignette focus toward center
UI Style:
Clean, minimal, not cluttered
Rounded panels, soft shadows
Hand-painted / illustrated look
Avoid:
Neon colors
Cyberpunk/futuristic UI
Overly complex HUD

📱 Layout (Mobile Portrait)
One-handed usability
Clear hierarchy:
Top = resources/status
Center = main content
Bottom = actions/navigation

🏠 Home Screen UI
Background: illustrated fantasy map (kingdom, forest, castle)
Top bar:
Gold, gems, player level
Center:
Path/map with stages (nodes connected by roads)
Bottom:
Large primary button: “Start / Continue”
Side buttons:
Quests, Events, Rewards (icon-based, medieval style)

🎁 Rewards Popup UI
Parchment-style panel (centered)
Title at top (fantasy font)
Grid of rewards:
Gold, EXP, items (icon + amount)
Buttons:
“Collect” (primary, glowing soft blue/gold)
“Exit” (secondary)
Subtle sparkle animation on rewards

🧙 Character / Equipment Screen
Center:
Full-body character (heroic pose)
Around character:
Equipment slots (weapon, armor, boots, etc.)
Bottom:
Stats bar (power, HP, attack)
Buttons:
Level Up
Equip Gear
Style:
Clean icons, gold frames, soft highlights

🃏 Card / Ability UI
Card frame:
Fantasy frame (wood/gold)
Elements:
Top-left: cost
Center: artwork
Bottom: short readable text
Keywords shown as small tags (e.g., Burn, Freeze)

⚔️ Combat UI (Simplified)
Top:
Wave counter
Base HP
Center:
Lanes/path with enemies
Bottom:
Card hand (5 cards max)
Energy indicator
Interaction:
Drag card to place
Keep UI minimal during combat

🧭 UX Principles
Simple navigation
Fast readability
Big touch targets
Minimal text, strong icons
Smooth transitions
No visual overload

✨ Extra Details
Soft glow on important buttons
Subtle particle effects (magic, gold shine)
Feedback animations (tap, collect, level up)
Consistent fantasy iconography
